# ProyectoIntegradorFinal

Este proyecto es una aplicación, similar a steam o epic store, donde los usuarios puede seleccionar sus juegos un comprarlos.
Con las funciones basicas de un portal de compra-venta juegos. 

# ¿Cómo será?

Este proyecto sería una base de datos de juegos, los usuarios podrán seleccionar y comprar el juego que deseen,
es evidente que los usuarios estarán almacenados en la base de datos. 

Los usuarios tendrán una cartera y un carrito para almacenar los juegos que van a comprar como el dinero que hay en la cuenta.

# Necesidades detetadas.

Por lo visto, para los usuarios, el tener varios portales de juegos no es problema, ádemas,
es bueno que haya más de una api como está, para que no haya solo una empresa ofreciendo 
los juegos.
